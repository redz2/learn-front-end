# 动画原理
什么是动画？
工作以来，处理过css、js、canvas、骨骼动画，这些背后的原理都是通过把每一帧（瞬间）的静止图像组合起来，以一定频率（速率）播放这些图像形成动画。
详细一点解析：动画可以拆分成每一帧，当前帧（静态）图像的属性数据或者形态（形状）的变更，把这样很多帧连贯起来，就形成动画。
